# Technical Labs Overview

This directory contains hands-on IT support and systems administration labs designed to simulate real-world scenarios commonly encountered in entry-level IT and help desk roles.

Each lab focuses on a specific technical domain and includes documentation of the environment, actions performed, verification steps, and key takeaways.

---

## Lab Categories

### OS Installation
**Location:** `labs/os-installation/`

Covers operating system deployment and validation in a cloud-based environment. Focus areas include system verification, networking confirmation, and remote access using SSH.

**Key Skills:**
- Linux system verification
- SSH connectivity
- Cloud VM validation
- Basic networking checks

---

### Security Basics
**Location:** `labs/security-basics/`

Focuses on user account management, authentication, least privilege enforcement, and controlled privilege escalation.

**Key Skills:**
- Linux user management
- Password handling
- Group-based permissions
- sudo access control
- Identity and access management fundamentals

---

### File Permissions
**Location:** `labs/file-permissions/`

Demonstrates file and directory permission management, ownership, and access troubleshooting.

**Key Skills:**
- chmod and chown
- File ownership concepts
- Permission troubleshooting
- Resolving access-related incidents

---

## Environment
- Platform: AWS EC2
- Operating System: Ubuntu Linux
- Access Method: SSH (Termius)
- Local Device: iPad-based workflow
