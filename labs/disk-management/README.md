# Disk Usage & Storage Management Lab

## Objective
Analyze disk usage and perform safe cleanup to prevent storage-related outages.

## Tools Used
- df
- du
- find
- apt
- snap

## Tasks Performed
- Checked filesystem usage
- Identified large directories and files
- Cleared apt caches
- Removed old snap revisions
- Verified post-cleanup disk health

## Outcome
Maintained system storage below risk thresholds and ensured continued stability.
