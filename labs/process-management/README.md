# Process Management & Monitoring Lab

## Objective
Monitor system performance and resolve high CPU usage.

## Tools Used
- uptime
- top
- ps
- kill

## Tasks Performed
- Checked system load averages
- Monitored active processes
- Identified high CPU usage process
- Terminated runaway process
- Verified system stability

## Scenario
Simulated a high CPU process using the `yes` command and resolved it using process management tools.

## Outcome
System performance restored and stabilized.
