# Server Security & Firewall Hardening Lab

## Objective
Implement host-based firewall rules to secure a cloud-hosted Linux system.

## Environment
- Platform: AWS EC2
- OS: Ubuntu
- Access: SSH (Termius)

## Tasks Performed
- Enabled UFW firewall
- Allowed SSH access
- Denied unnecessary ports
- Tested firewall enforcement
- Removed temporary rules

## Security Principles
- Defense in Depth
- Least Privilege
- Attack Surface Reduction

## Outcome
Successfully hardened the server against unauthorized network access.
