# Support Tickets Overview

This directory contains documented help desk tickets based on real-world
IT support scenarios encountered during hands-on labs.

Each ticket follows a standard incident documentation format, including
issue summary, impact, troubleshooting steps, root cause, resolution,
and prevention.

---

## Ticket Index

### Ticket 001 – EC2 SSH Connectivity Issue
Documents loss of remote access to a cloud-hosted system due to a security
group misconfiguration and the steps taken to restore connectivity.

---

### Ticket 002 – User Privilege Escalation
Documents a user account lacking required administrative privileges and the
process of verifying least privilege and granting controlled sudo access.

---

## Purpose

These tickets demonstrate:
- Troubleshooting methodology
- Clear written communication
- Root cause analysis
- Resolution and prevention practices

This documentation is intended to reflect professional help desk and
IT support workflows.
